## Variant: 沉浸写作台

### Design stance
把 Studio 当作一张安静的数字稿纸，技术字段退到抽屉，正文始终占据视觉中心。

### Key choices
- Layout: 可折叠文章库 + 居中纸张式编辑器 + 右侧设置抽屉
- Typography: 柔和无衬线中文标题，正文使用舒展行高
- Color: 暖白纸面、低饱和蓝灰、正式双主题
- Interaction: 写作/真实预览切换、文章设置抽屉、发布检查弹窗、恢复副本状态

### Trade-offs
- Strong at: 中文长文、低干扰写作、日常快速发布
- Weak at: 同时比较大量文章状态和批量管理

### Best for
- 个人博客作者以一篇文章为中心完成写作、预览和发布
