## Variant: 引导式发布台

### Design stance
把“发博客”做成清晰的内容、预览、检查、发布四步流程，让用户始终知道下一步是什么。

### Key choices
- Layout: 文章库 + Markdown 编辑器 + 常驻真实站点预览
- Typography: 界面紧凑、正文编辑使用等宽字体、预览保持网站阅读感
- Color: 深海蓝黑工作台、低饱和蓝紫与绿色状态反馈
- Interaction: 草稿筛选、桌面/手机预览、专注写作、发布检查侧板

### Trade-offs
- Strong at: 发布确定性、即时预览、状态清晰
- Weak at: 三栏在小屏幕上更拥挤，纯写作时工具感更强

### Best for
- 经常需要检查最终页面、希望按固定流程发布文章的个人作者
